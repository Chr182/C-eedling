# C-eedling 
A C-Based Pomodoro Timer for Improving Student Engagement and Academic Persistence
## Description
This program aims to improve study retention by implementing an incentive in the form of a C-eedling™ that students can help grow by racking up study minutes. This project aims to tackle the fourth SDG (Quality Education) with the topic of 
Improving Student Engagement and Academic Persistence through the Pomodoro Technique.
